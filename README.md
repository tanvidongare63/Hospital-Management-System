# 🏥 Hospital Management System

## 📌 Project Overview

The **Hospital Management System** is a mini-project developed to simulate basic hospital operations such as managing patient data and system workflows. This project combines a simple web-based frontend with core logic implemented in C++, demonstrating the practical use of Data Structures and Algorithms (DSA).

This project is primarily focused on **learning and academic purposes**, showcasing how real-world systems can be modeled using programming concepts.

---

## 🚀 Features

* Simple and user-friendly interface
* Basic hospital workflow simulation
* Efficient data handling using C++ (DSA concepts)
* Structured and modular code design
* Lightweight and easy to run

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend Logic:** C++
* **Concepts Used:** Data Structures and Algorithms (DSA)

---

## 📂 Project Structure

```
dsal/
│
├── DSAL.cpp        → Main C++ source code (core logic)
├── hospital.exe    → Compiled executable file
├── index.html      → Main frontend page
├── script.js       → JavaScript functionality
├── styles.css      → Styling file
├── README.md       → Project documentation
```

---

## ▶️ How to Run the Project

### 🔹 Run Frontend (UI)

1. Open the project folder
2. Double-click on `index.html`
3. The project will open in your default browser

---

### 🔹 Run C++ Program (Core Logic)

#### Step 1: Open terminal in project folder

#### Step 2: Compile (if `.exe` not available)

```bash
g++ DSAL.cpp -o hospital
```

#### Step 3: Run program

```bash
hospital.exe
```
